React Native Todo App

This TODO app, streamlines task management and enhances productivity through intuitive task creation, organization, and completion features. Employing React Native, Expo, and React Navigation, I developed Taskly's frontend, integrating components like FlatList for efficient task rendering and React Native Elements for UI elements. Additionally, I utilized async-storage for data persistence, ensuring seamless user experience across sessions. One challenge was implementing task organization by time and date, including categorizing tasks into "completed" and "tasks due soon" categories. This was addressed by leveraging FlatList's flexibility to dynamically render tasks based on their completion status and deadline, ensuring effective task tracking and prioritization. The The use of different colors for each task, including red for completed tasks, enhances visual distinction and aids users in quickly identifying and prioritizing tasks.

Exercise App Wireframe Design

TThe project aimed to design wireframes for an exercise app, featuring a main screen displaying three exercises, along with distinct screens for duration and repetition exercises, each providing intuitive controls for effective user interaction. Utilizing a design tool like Figma, I focused on understanding user interface design principles, layout structuring, and visual hierarchy establishment. One challenge encountered was designing distinct screens for duration and repetition exercises while ensuring consistency in user experience. This was addressed by maintaining a clear layout and incorporating intuitive controls specific to each exercise type. In terms of design decisions, I opted for a clean and minimalist layout to prioritize user focus on exercise details, prominently displaying exercise names for quick identification and incorporating intuitive controls for ease of use. Additionally, I chose a color scheme with subtle accents to enhance visual appeal without distracting from the main content.

Exercise App

The purpose of this project is to create a simple Exercise Tracker application where users can select exercises from a menu and track their repetitions or duration. React was used to build the application, and functional components were implemented too create Repetition and Duration exercise functionalities. One problem was updating each component based on user interactions. This was solved by using React's state management system to track and update values. A blue color scheme was chosen for its calming, universally appealing, and professional qualities, aligning with the application's purpose of promoting user trust and readability in tracking exercise routines.

FlashCard Study App Design

This project is a native mobile application designed to facilitate efficient and engaging flashcard studying. With features like flashcard creation, editing, deletion, and randomized display, users can enhance their learning experience and retention through interactive study sessions. Leveraging Expo and React Native Elements, the app frontend was designed with a focus on intuitive UI design and seamless navigation. Figma was utilized for visual design, incorporating font, color, and button styles to enhance user experience and aesthetic appeal. One challenge was implementing randomized display of flashcards while ensuring each card is shown only once. This was addressed by designing a UI flow that prioritizes efficient navigation and intuitive controls, allowing users to seamlessly access flashcards and track their study progress. The design proritizes simplicity, with a clean interface or edtiing, creating, and deleting flashcards.Each flashcard features a prompt on one side and an answer on the other, enhancing study effectiveness

Todo App Wireframe Design

Taskly, designed in Figma, is a streamlined TODO app aimed at simplifying task management and user authentication through an intuitive interface. Utilizing Figma, I crafted wireframes and refined the final design, applying user interface design principles, layout structuring, and color psychology to enhance the user experience. Specifically, I implemented features such as task creation, deletion, deadline setting, user authentication, and task completion functionalities. A key challenge was seamlessly integrating essential functionalities while maintaining a clean and intuitive layout. This was resolved through meticulous organization of the interface and iterative usability testing. Taskly's layout prioritizes ease of use and visual clarity, with a teal and gray color scheme chosen for its calming and professional aesthetic.
